内联样式：
	直接对HTML标签使用style属性
内部样式：
	写在<head>标签中，使用<style>声明
外部样式：
	语法：
		<link type="text/css" rel="stylesheet" href="外部样式文件">
导入外部样式：
	与<link>相似  语法不同  在<style>中使用
		@import"样式文件"或@import url("样式文件")

优先级：内联样式->内部样式->导入外部样式->外部样式

----------------------------------------------------------------------------------
选择器：
	元素选择器：标签名{ 样式k:v }

	类选择器：.classValue{ 样式k:v }	标签名.classValue{ 样式k:v }

	id选择器：#idValue{ 样式k:v }		标签名#idValue{ 样式k:v }

	选择器组合：一次性声明多个元素样式  逗号隔开

	选择器嵌套：标签布局嵌套 使用空格隔开
